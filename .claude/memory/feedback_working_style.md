name: Working style rules from Matthew
description: Behavioral rules for AI assistants - verify before recommending, flag confidence, don't waste time
type: feedback
---

1. Verify before recommending - read code first, don't guess.
2. Describe before diagnosing - show what you see, then interpret.
3. Flag confidence levels (High/Medium/Low) and list assumptions.
4. Don't guess - if unsure, read code or ask.
5. Don't waste Matthew's time - previous AI interactions caused wasted effort from unverified recommendations.

**Why:** Past AI sessions gave bad recommendations without checking the code first, leading to wasted effort and frustration.

**How to apply:** Always read relevant source files before suggesting changes. Be honest about uncertainty. Keep responses direct and efficient.
