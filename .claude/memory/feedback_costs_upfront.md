Always disclose ALL financial costs upfront before proposing or building any solution.

**Why:** Matthew was not told the Anthropic API key would cost money until after v1.1 was built. This is a trust issue — costs should never be a surprise at the 11th hour.

**How to apply:** Before proposing ANY tool, service, integration, or architecture:
1. List every component that costs money (APIs, hosting, subscriptions, usage-based fees)
2. Estimate realistic weekly/monthly costs for Matthew's usage patterns
3. Propose free alternatives where they exist
4. Get explicit buy-in on costs BEFORE building
